# 📰 Learning to Read Politics

## 🗂️ Dataset

This project uses the [**News Articles for Political Bias Classification**](https://www.kaggle.com/datasets/gandpablo/news-articles-for-political-bias-classification) dataset from Kaggle.

<div style="background-color: rgba(107, 98, 90, 0.2); padding: 20px; border-radius: 10px; margin-bottom: 20px;">
<h2>Setup Instructions</h2>
<p>
1. Download the dataset from the link above.
2. Extract the downloaded files.
3. Locate the file `bias_clean.csv`.
4. Create the following folder structure in the project directory (if it does not already exist): data/raw
5. Place `bias_clean` inside: data/raw/bias_clean.csv 
</p>
</div>

Notes
- The dataset is not included in this repository due to its size
- The code expects the file to be located exactly at `data/raw/bias_clean.csv`
