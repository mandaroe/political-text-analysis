# 📰 Learning to Read Politics

## 🗂️ Dataset

This project uses the [News Articles for Political Bias Classification](https://www.kaggle.com/datasets/gandpablo/news-articles-for-political-bias-classification) dataset from Kaggle.

| ⚙️ Setup |
|----------------------|
| 1. Download the dataset |
| 2. Extract the files |
| 3. Locate `bias_clean.csv` |
| 4. Create `data/raw` |
| 5. Place file in `data/raw/bias_clean.csv` |

**Notes**
- Dataset not included due to size  
- Expected path: `data/raw/bias_clean.csv`

## Instructions
```python
python train.py --task bias
python train.py --task topic
```
